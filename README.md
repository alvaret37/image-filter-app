# 🖼️ Image Filter App

Aplique filtros de imagem como preto e branco, blur, cartoon e mais.

## 📦 Estrutura
```
image_filter_app/
├── app/
│   └── main.py
├── images/
│   ├── input/
│   └── output/
├── tests/
├── .gitignore
├── requirements.txt
└── README.md
```

## ▶️ Como rodar

Coloque uma imagem em `images/input/sample.jpg` e execute:

```bash
python app/main.py
```

## ✅ Resultado esperado

A imagem será salva em P&B na pasta `output`.
